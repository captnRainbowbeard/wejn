Water Resources Metering Real-time Monitoring System Solution

Nov 15, 2019

System Overview

The state-controlled water resource monitoring system is an important guarantee for real-time control of the dynamics of water, water, water and drainage, and protects the accuracy, scientificity and refinement of first-hand information. Based on information automation data collection and transmission, equipped with advanced instrumentation equipment, improve the automation level of information collection, transmission and processing, form a complete information collection system, and provide timely and accurate information support for water resources management.

Status description

For traditional remote meter reading, the following problems exist:

Problem 1: Lack of dedicated system management software

Question 2: The meter reading cycle is long and the data is summarized slowly.

Question 3: The failure of the table cannot be discovered in time, and the water volume will be missed.

Question 4: If there is a dispute in the post work, it cannot provide an accurate basis.

The goal of building

The remote monitoring system for water intake (flow) is designed for various measurement and accurate problems existing in the current water abstraction process. A remote monitoring system for water intake (flow) is established in the relevant units, and various types of smart water meters and ultrasonic flow meters are used. Intelligent metering, combined with computer and communication network, realizes real-time monitoring of water intake by water users.

The technology realizes real-time online monitoring, stores and organizes the measured data, and manages and controls the water intake plan, provides reliable data support for various post-workers' work, and promotes sustainable use of water resources and water conservation.

System composition

The monitoring terminal box is installed at the monitoring point, and the telemetry terminal is installed therein, and the telemetry terminal is directly connected with the instrument such as the flow meter. The wireless data transmission module of the telemetry terminal automatically logs into the GPRS/4G network of the mobile communication after power-on, and automatically reports the monitoring data (such as traffic) to the data management system server of the monitoring center according to the set test time interval.

The monitoring center computer room is equipped with a dedicated server computer, and the server installs the groundwater level remote monitoring system software and server system. Through RTU, the real-time monitoring and management of the data collected and received by the monitoring point is realized.
