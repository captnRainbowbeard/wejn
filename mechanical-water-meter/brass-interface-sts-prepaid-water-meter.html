Product Details

STS prepaid water meter is a water meter that users can recharge by entering a 20-digit recharge code through CIU


The sts prepaid water meter's feature

1. Alarm function: If illegal disassembly is encountered, the alarm function will be activated

2. Data docking: professional software for data docking

3. Humanized settings: the software can set up multiple languages

4. More flexible rate setting

5. Quick query, quick recharge

6. Diversified payment methods, making payment more convenient

7. 20-digit Token, higher security, according to the table to generate the recharge code, one table one code

8. Split-type built-in lora module, configurable concentrator component pipe network monitoring system
